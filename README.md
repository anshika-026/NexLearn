# NexLearn: AI Tutor

An AI-powered study tutor built with Google Gemini and Streamlit.

## Features
- Ask subject questions
- Generate quizzes
- Get topic explanations
- Chat history

## Tech Stack
- Python
- Streamlit
- Google Gemini AI

## How to Run
1. Clone the repo
2. Install dependencies: pip install -r requirements.txt
3. Add your GOOGLE_API_KEY to .env file
4. Run: streamlit run app.py
